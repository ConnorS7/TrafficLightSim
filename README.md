# OOPFinalProject

## Maze Generation and Solving Application
### Project Overview
A Java-based maze generation and solving application that implements multiple algorithms to create and navigate mazes of varying complexity. The program features a Java 2D visualization that displays the maze structure and animates the solving process in real-time.
